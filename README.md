# SWE201 – C# Console Application Projesi  
### Burak Onur Can Çörtük – 215060033

Bu proje, SWE201 Laboratuvar dersi kapsamında istenen C# Console Application ödevi için hazırlanmıştır.  
Proje **Visual Studio** kullanılarak geliştirilmiştir.

---

## 1. Sınıf Dosyası (Ogrenci.cs)

Bu dosya içinde **Ogrenci** adında bir sınıf tanımlanmıştır.  
Sınıfta aşağıdaki yapılar bulunmaktadır:

- İki adet üye değişken:
  - `ad`
  - `yas`
- Otomatik özellikler:
  - `Bolum`
  - `Ortalama`
- Parametresiz kurucu fonksiyonu
- Parametre alan kurucu fonksiyonu
- Ortalama güncelleme fonksiyonu
- `ToString()` override

---

## 2. Program Dosyası (Program.cs)

Bu dosyada `Main` fonksiyonu içinde:

- Ogrenci sınıfından nesneler oluşturulmuş,
- Kurucular test edilmiş,
- Fonksiyon çağrıları yapılmış,
- Sonuçlar konsola yazdırılmıştır.

---

## Proje İçeriği

```
Ogrenci.cs
Program.cs
README.md
```

Ödev yönergelerinde istendiği gibi yalnızca `.cs` dosyaları ve README eklenmiştir.

---

## Çalıştırma Bilgileri

Proje **Visual Studio** ortamında hazırlanmıştır ve standart bir C# Console Application yapısındadır.

Çalıştırmak için:

1. Visual Studio’da projeyi açın  
2. **Start / Başlat** tuşuna basın  

---

## Geliştirme Adımları

1. Ogrenci sınıfı oluşturuldu  
2. Üye değişkenleri ve özellikler eklendi  
3. Kurucular yazıldı  
4. Ortalama güncelleme fonksiyonu yazıldı  
5. ToString() override edildi  
6. Program.cs içinde test amaçlı nesneler oluşturuldu  
7. Konsol çıktı kontrolü yapılarak proje tamamlandı
